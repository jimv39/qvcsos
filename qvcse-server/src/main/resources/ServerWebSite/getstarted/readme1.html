QVCS Enterprise -- Installation and Getting Started...

Before you can use QVCS-Enterprise, you must first download and install an appropriate Java virtual machine. We have only tested with Oracle's JVMs. Oracle's download site is available at this URL:

https://www.oracle.com/java/technologies/

We have tested with Java 8.

Build your own

QVCS-Enterprise is open source. You can build your own copy of the server and client applications. See https://github.com/jimv39/qvcsos for some guidance on getting started. After building the applications, you can relocate the created testenterprise directory tree to where ever makes sense for you.
