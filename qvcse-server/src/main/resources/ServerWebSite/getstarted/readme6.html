QVCS Enterprise -- Installation and Getting Started...

Step 10: Launch the GUI client:

For Linux or MacOS users, open a new shell window, and execute the client.sh shell script (you may need to chmod +x to make it executable).

The client application should start up. You'll probably want to resize the main window, and some of the splitter bars to make things more usable.

On the left side of the screen, you'll see a list of servers known to this client; to the right is the file pane, which will show any files associated with the currently selected node on the left side of the screen. On the bottom right is a tabbed pane which will provide various version control information about any selected files.

Step 11: Login to the Server:

Select the server node that you defined in Step 4. The Server login dialog will popup. Enter a username and password that you defined in Step 7.

The server should permit you to login, provided you supply a correct username/password combination.

If you expand the server node on the left side of the screen, you should see the project that you defined in Step 8.
