Server command line

The Linux/MacOS command line for the server supplied with the product distribution looks like:

java -jar qvcse-server-VERSION.jar "$QVCS_HOME" 9889 9890 9080

This command line is found in the server.sh shell script.

The 3 numbers immediately following that value identify the ports that the server will use.

  • The 1st number (9889) identifies the port the server listens on for connections from the QVCS-Enterprise client application. This is the port that you should point your client to.
  • The 2nd number (9890) identifies the port the server listens on for connections from the admin tool.
  • The 3rd number (9080) identifies the port that the server listens on for http connections -- i.e. the last number is the port number used for the simple web server included with the server. You should be able to point your favorite web browser to that port to see the web pages served by the server.
For the source code, see https://github.com/jimv39/qvcsos
