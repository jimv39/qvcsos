Client application command line

The Linux/MacOS command line for the client application supplied with the product distribution looks like:

java -Xmx256M -jar qvcse-gui-VERSION.jar "$QVCS_HOME"

This command line is found in the client.sh shell script.

The purpose of the argument at the end of the command line is to identify the 'install' directory for the client application.
For the source code, see https://github.com/jimv39/qvcsos
