QVCSOS-Enterprise User Manual

This is the user manual for QVCSOS-Enterprise.

QVCSOS-Enterprise is a simple to use version control product that allows you to capture snapshots of the files you work with. It is implemented as a client/server application.

This release of QVCSOS-Enterprise provides these version control features:

  • Add files to version control.
  • Check in file revisions to capture changes to the files that you have under version control.
  • Get file revisions from the server so you can recover past work if required.
  • Visually compare your work to past revisions with a built-in visual compare tool.
  • Visually compare your work to past revisions using any external visual compare tool that can be launched from the command line.
  • See at a glance which of your files have changed or are out of date.
  • See all the files of a project's directory tree in a single file list pane.
  • Merge (automatically or visually) two separate edits to the same file.
  • Project level branching support. Create Feature branches, Release branches, and Read-only tag based branches.
  • Simple reporting based on file and revision filter criteria.
  • Rename a file.
  • Move a file from one project directory to another directory in the same project.
  • Delete a directory.
  • And more...

QVCSOS-Enterprise includes a separate administrator's application that you use to define projects, users, and user roles for the defined projects.

QVCSOS-Enterprise also includes a custom ANT task that can be used to provide simple version control behaviors from within an ANT build file.

For the source code, see https://github.com/jimv39/qvcsos
