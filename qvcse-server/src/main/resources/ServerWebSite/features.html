QVCSOS-Enterprise Feature Summary

QVCSOS-Enterprise is a 100% Java version control system. QVCSOS-Enterprise is a client/server application specifically designed for use over the Internet. QVCSOS-Enterprise works well for both geographically dispersed teams and for local teams.

QVCSOS-Enterprise Features:

  • Store/retrieve file revisions.
  • Keep track of who changed what, and when they changed it.
  • Apply a tag to mark the state of your project, and later use that tag to create a read-only branch based on that tag.
  • Control modification access to files at the project level.
  • Use built-in visual compare tool to compare files to earlier revisions or compare revisions to other revisions.
  • Use built-in visual merge tool to merge changes from two separate edits to the same file.
  • Create feature branches, release branches, and read-only tag based branches.
  • Create a variety of reports including:
    • report all revisions created after a given commit id.
    • report all revisions created before a given commit id.
    • report revisions created between two different commit ids.
    • report all revisions created by a single individual, or set of individuals
    • report all revisions created by a given commit.
    • report any AND'd combination of the above
  • Show at a glance all local files that are different from the latest checked in revision.
  • Create a .qvcsosignore. file on the client to tell the client which files should be ignored.
  • Define named file filters to easily filter the set of files that will get displayed.
  • View all the files of a single project within a single file list.
  • Move a file from one directory to another within a project's branch.
  • Delete a directory (privileged users only).
  • Apply a tag to a branch to mark some significant development milestone.
  • Create a read-write feature branch.
  • Create a read-write release branch.
  • Create a read-only tag based branch.
  • Manage user roles at the project level using a separate, secure admin tool.
  • Use custom Ant task to perform bulk operations.
  • Connect to the QVCSOS-Enterprise server via the Internet, VPN, LAN, or locally.
  • And more ...

What's Included:

The QVCSOS-Enterprise product includes the server application, a client application, and a separate admin tool.

These applications can be deployed on a single machine (when working alone, for example), or on separate machines, as required.

On-line documentation is included with the server and is 'published' by the server via a simple built-in web server.

QVCSOS-Enterprise System Requirements:

  • Java 17 or later.
  • A PostgreSQL server; release 13.
